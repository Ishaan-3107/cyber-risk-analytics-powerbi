# 📊 Cyber Risk & Exception Analytics – Power BI

---

## 📌 Overview

This repository contains a structured collection of Power BI dashboards focused on:

- Security exception monitoring  
- Vulnerability analysis  
- Risk posture tracking  
- Remediation trend analysis  

The objective of this project is to design analytical dashboards that provide visibility into exception lifecycle management and vulnerability exposure using structured data modeling and DAX calculations.

---

## 🎯 Project Objectives

The reports demonstrate:

- ⭐ Star schema modeling  
- 📅 Time intelligence calculations  
- ⏳ Aging analysis  
- 📈 Risk distribution analysis  
- 📊 KPI-driven dashboard design  
- 🔎 Interactive filtering & drill-down capability  

> ⚠️ All datasets used in this repository are anonymized and prepared strictly for demonstration purposes.

---

## 🏢 Business Context

Organizations often maintain:

- Exception registers  
- Vulnerability inventories  
- Risk classification logs  

Without structured reporting, it becomes difficult to:

- Track how long exceptions remain open  
- Monitor recurring exception patterns  
- Measure closure performance  
- Identify high-risk vulnerabilities  
- Analyze trends across time periods  

This project addresses these analytical needs using Power BI.

## 🏗 Data Modeling Approach

The dashboards follow a **Star Schema architecture**.

### 🔹 Fact Tables
- `Exceptions`
- `Vulnerabilities`

### 🔹 Dimension Tables
- `Date`
- `Business Unit / Department`
