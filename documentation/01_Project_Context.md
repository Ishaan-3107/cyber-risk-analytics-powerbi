# Project Context

The purpose of this project is to create structured analytical dashboards for exception management and vulnerability monitoring.

Exception management is a critical part of governance and risk oversight. Without structured reporting, tracking open items, aging trends, and closure performance becomes difficult.

Similarly, vulnerability tracking requires monitoring severity distribution and remediation timelines.

This project focuses on building data models and dashboards that provide:

- Clear visibility into approved vs rejected items
- Aging analysis for risk prioritization
- Trend tracking over time
- Severity-based segmentation
- Interactive filtering capabilities

The dashboards are designed to be intuitive and performance-optimized using Power BI best practices.
