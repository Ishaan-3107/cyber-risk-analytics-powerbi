# Dashboard Design Decisions

The dashboards were designed with clarity and usability as priorities.

## KPI Cards
Used for high-level summaries such as:
- Total exceptions/vulnerabilities
- Critical vulnerabilities
- Approval rate
- Rejection rate
- Top host
- Top port

## Bar Charts
Used for:
- Status comparison
- Severity distribution
- Asset distribution
- Host distribution

## Line Charts
Used for:
- Monthly trend analysis
- Discovery vs remediation tracking


## Slicers
Limited to essential filters:
- Date
- Status
- Risk Category
- Request type
- Department


Color logic:
- Red for critical risk
- Orange for high
